Android App für die F-Droid Build Status API.

Funktionen

- Zeigt fehlgeschlagene und erfolgreiche Builds
- Zeigt das Build-Protokoll im Detailbildschirm
- Zeigt veröffentlichte Versionen einer App an
- Zeigt an, ob eine App ein Update benötigt oder deaktiviert ist
- Markieren Sie Apps als Favoriten für die Sortierung
- Benachrichtigung über einen neuen Build-Status, optional nur für Ihre bevorzugten Apps

Verwendete F-Droid API:

Die App verwendet die folgenden F-Droid API Endpunkte:

- running.json: der aktuell laufende F-Droid Buildserver-Prozess, https://f-droid.org/repo/status/running.json
- build.json: der letzte abgeschlossene F-Droid Buildserver Build-Zyklus, https://f-droid.org/repo/status/build.json
- update.json: die letzte Aktualisierungsprüfung des F-Droid Buildservers, https://f-droid.org/repo/status/update.json
- build.log: https://f-droid.org/repo/{id}_{versionCode}.log.gz
- veröffentlichte Paketinformationen: https://f-droid.org/api/v1/packages/{id}, Beispiel https://f-droid.org/api/v1/packages/org.fdroid.fdroid
