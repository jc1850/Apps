使用 F-Droid 构建状态 API 的 Android 应用程序。

特点

- 显示失败和成功的构建
- 在屏幕上显示详细构建日志
- 显示一个应用程序的已发布版本
- 显示一个应用程序是否需要更新或被禁用
- 将应用程序标记为已收藏，以便进行排序
- 通知新的构建状态，可选择只通知你收藏的应用程序

使用的 F-Droid API 。

此应用程序使用以下 F-Droid API 端点。

- running.json：当前运行的F-Droid构建服务器进程，https://f-droid.org/repo/status/running.json
- build.json：最后完成的F-Droid buildserver构建周期，https://f-droid.org/repo/status/build.json
- update.json：F-Droid构建服务器的最后一次更新检查，https://f-droid.org/repo/status/update.json
- build.log：https://f-droid.org/repo/{id}_{versionCode}.log.gz
- 发布的软件包信息：https://f-droid.org/api/v1/packages/{id}, 示例 https://f-droid.org/api/v1/packages/org.fdroid.fdroid
