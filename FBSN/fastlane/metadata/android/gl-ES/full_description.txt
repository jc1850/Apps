Aplicación Android para o estado de compilación da API de F-Droid.

Características

- Amosa compilacións fallidas e exitosas
- Amosa o rexistro de compilación na pantalla de detalles
- Amosa as versións publicadas dunha aplicación
- Amosa se unha aplicación precisa dunha actualización ou está deshabilitada
- Marca as aplicacións coma favoritas para ordenar
- Notifica sobre novos estados de compilación, opcionalmente só para as túas aplicacións favoritas

A API de F-Doid empregou:

A aplicación empregou os seguintes endpoints da API de F-Droid:

- running.json: o proceso actual de buildserver de F-Droid en execución, https://f-droid.org/repo/status/running.json
- build.json: o último ciclo de compilación completo do buildserver de F-Droid, https://f-droid.org/repo/status/build.json
- update.json: a última comprobación de actualización do buildserver de F-Droid, https://f-droid.org/repo/status/update.json
- build.log: https://f-droid.org/repo/{id}_{versionCode}.log.gz
- información dos paquetes publicados: https://f-droid.org/api/v1/packages/{id}, example https://f-droid.org/api/v1/packages/org.fdroid.fdroid
