Aip android don api stádas tógála F-Droid.

Gnéithe

- Taispeáin theip agus tógann rathúil
- Taispeáin an scáileán mion logáil isteach
- Léiríonn leaganacha foilsithe d ' aip
- Taispeáin an bhfuil nuashonrú de dhíth ar aip nó an bhfuil sí faoi mhíchumas
- Marcáil apps mar is fearr leat le haghaidh sórtála
- Fógra a thabhairt faoi stádas tógála nua, de rogha amháin de do apps is fearr leat

F-Droid API a úsáidtear:

Úsáideann an Aip na críochphointí API F-Droid seo A Leanas:

- ag running.json: an próiseas reatha Tógála F-Droid, https://f-droid.org/repo/status/running.json
- build.json: an ceann deireanach críochnaithe f-Droid tógálaí timthriall tógála, https://f-droid.org/repo/status/build.json
- update.json: an seiceáil nuashonraithe is déanaí den fhreastalaí Tógála F-Droid, https://f-droid.org/repo/status/update.json
- build.log: https://f-droid.org/repo/{id}_{versionCode}.log.gz
- eolas faoin phacáiste foilsithe: https://f-droid.org/api/v1/packages / {id}, sampla https://f-droid.org/api/v1/packages/org.fdroid.fdroid
