Приложение для Android для API статуса сборки F-Droid.

Особенности

- Показывает неудачные и успешные сборки
- Показывает журнал сборки на подробном экране
- Показывает опубликованные версии приложения
- Показывает, нуждается ли приложение в обновлении или оно отключено
- Отметьте приложения как избранные для сортировки
- Уведомлять о статусе новой сборки, необязательно только о ваших любимых приложениях

Используемый API F-Droid:

Приложение использует следующие конечные точки F-Droid API:

- running.json: текущий запущенный процесс F-Droid buildserver, https://f-droid.org/repo/status/running.json
- build.json: последний завершенный цикл сборки F-Droid buildserver, https://f-droid.org/repo/status/build.json
- update.json: последняя проверка обновления сервера сборки F-Droid, https://f-droid.org/repo/status/update.json
- build.log: https://f-droid.org/repo /{идентификатор}_{код версии}.log.gz
- опубликованная информация о пакете: https://f-droid.org/api/v1/packages /{идентификатор}, пример https://f-droid.org/api/v1/packages/org.fdroid.fdroid
