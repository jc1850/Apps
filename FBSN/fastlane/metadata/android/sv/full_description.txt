Android App för F-Droid byggstatus API.

Funktioner

- Visa misslyckade och lyckade byggen
- Visa bygglogg i detaljerad vy
- Visa publicerad version av en app
- Visa om en app behöver en uppdatering eller är inaktiverad
- Markera appar som favoriter för sortering
- Notifiera om nya byggstatusar, möjligen bara dina favoritappar

F-Droid API användning:

Appen använder följande F-Droid API ändpunkter:

- running.json: den nuvarande körningen av byggserver processen, https://f-droid.org/repo/status/running.json
- build.json: den senaste färdigställde F-Droid byggserver byggcykel, https://f-droid.org/repo/status/build.json
- update.json: den senaste uppdateringskontrollen av F-Droid byggserver, https://f-droid.org/repo/status/update.json
- build.log: https://f-droid.org/repo/{id}_{versionCode}.log.gz
- published package information: https://f-droid.org/api/v1/packages/{id}, till exempel https://f-droid.org/api/v1/packages/org.fdroid.fdroid
