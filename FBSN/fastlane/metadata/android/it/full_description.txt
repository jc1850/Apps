App Android per l'API dello stato di sviluppo di F-Droid.

Caratteristiche

- Mostra le build fallite e quelle riuscite
- Mostra il log della compilazione nella schermata di dettaglio
- Mostra le versioni pubblicate di un'app
- Mostra se un'app ha bisogno di un aggiornamento o Ã¨ disabilitata
- Segna le app come preferite per l'ordinamento
- Notifica lo stato di una nuova build, opzionalmente solo per le tue app preferite

F-Droid API utilizzati:

L'app utilizza i seguenti endpoint F-Droid API:

- running.json: l'attuale processo di buildserver F-Droid in esecuzione, https://f-droid.org/repo/status/running.json
- build.json: l'ultimo ciclo di buildserver F-Droid completato, https://f-droid.org/repo/status/build.json
- update.json: l'ultimo controllo di aggiornamento del buildserver F-Droid, https://f-droid.org/repo/status/update.json
- build.log: https://f-droid.org/repo/{id}_{versionCode}.log.gz
- informazioni sul pacchetto pubblicato: https://f-droid.org/api/v1/packages/{id}, esempio https://f-droid.org/api/v1/packages/org.fdroid.fdroid
