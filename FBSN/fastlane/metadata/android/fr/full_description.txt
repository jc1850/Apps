Application Android pour l'API d'état de construction F-Droid.

Caractéristiques

- Affiche les constructions échouées et réussies
- Affiche le journal de construction dans un écran détaillé
- Affiche les versions publiées d'une application
- Indique si une application a besoin d'une mise à jour ou si elle est désactivée.
- Marquez les applications comme favorites pour les trier
- Notification de l'état de la nouvelle version, optionellement uniquement pour vos applications favorites.

API F-Droid utilisée :

L'application utilise les points de terminaison de l'API F-Droid suivants :

- running.json : le processus de serveur de construction F-Droid en cours, https://f-droid.org/repo/status/running.json
- build.json : le dernier cycle de construction du serveur de construction F-Droid terminé, https://f-droid.org/repo/status/build.json
- update.json : la dernière vérification de mise à jour du serveur de construction F-Droid, https://f-droid.org/repo/status/update.json
- build.log : https://f-droid.org/repo/{id}_{versionCode}.log.gz
- informations sur les paquets publiés : https://f-droid.org/api/v1/packages/{id}, exemple https://f-droid.org/api/v1/packages/org.fdroid.fdroid
